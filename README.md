# Water-Prop-Fivem
These props were created to help all creators who use heavy props only to use water particles.
These props, due to their small size and invisible variant, can be hidden anywhere.

All 10 props have different water and invisible variant.
You can see all  here:  

https://youtu.be/L5wz3JmRg30
